# Whirlpool Framework

A lightweight, component-based JavaScript framework for building modular web applications with event-driven architecture.

## Project Home

For full documentation, examples, and guides, please visit the [Whirlpool project homepage](https://github.com/lulurun/whirlpool).

## Quick Start

```bash
# Clone the repository
git clone https://github.com/lulurun/whirlpool.git
cd whirlpool

# Build the framework
npm install
npm run build

# Try an example
cd examples/01-dynamic-form
npm install
npm start
```

The app will open at http://localhost:8080
