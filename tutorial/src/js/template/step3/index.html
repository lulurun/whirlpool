How it works:
  1. Click the button in the Event Publisher component
  2. It emits an event: app.ev.emit('button.clicked', data, this)
  3. The Message Display and Event Counter components subscribe to this event
  4. Both components receive the event and update automatically

Key Point: The components don't know about each other. They only know about the event name 'button.clicked'. This is decoupled architecture.
